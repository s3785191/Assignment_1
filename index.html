My Project Idea

Overview

The electronic reminder product is a system which can be used by anyone. Whether they are an individual or involved in a group or team-based environment this product will assist with the organisation and awareness of issues to be resolved and tasks to be completed. Also, issues that need a solution can be reported and everyone will be notified so that it can be resolved quickly - whether it be a workplace issue or with a task that is currently being worked on. This product will be a software solution.

Motivation

This will be useful mostly for teams/groups that have a task to complete within a timeframe or companies that rely heavily on teamwork. The group can use this product to manage their tasks and assist each other so that tasks/projects can be completed on time. Businesses can use the product for the same reasons, however, they can also use the ‘report an issue’ functionality which will create and spread awareness across to everyone about issues that need to resolved. For businesses, completing a certain task on time could mean more sales/profits rather than a loss in profits and, potentially, consumers.

Description

Tasks - This product allows individuals or teams to create a ‘to-do-list’. The task’s description and its progress status would be sent to everyone in the team/group on a daily/weekly basis. This is so everyone knows what they need to do, what others need to do and when a task needs to be completed. The ‘tasks’ will contain the following attributes:

  • Name of task
  • Category/Department
  • People responsible for the task
  • Task details (what needs to be done)
  • Progress status
  • Recommended completion date

Reporting Issues – The product also allows people to report issues within a workplace, such as a hazard, so that everyone will know about it or report that they are having difficulty with a task. Everyone will be notified when an issue is reported and when an issue is resolved. The ‘issues’ will contain the following attributes:

  • Name of issue
  • Date reported
  • Person who reported the issue
  • Details of the issue
  • Status (unresolved/resolved)

Users will have the option to select either create a task or report an issue. Once the user has selected either option they must fill in the form with all the necessary details before submitting it. After the submission, the form will be formatted into a suitable format and emails will be sent out to notify those involved.

Users will need to update the status of tasks they are involved in so that everyone else understands the progress of the task. However, a concern for this system is that users may forget or simply not update the status of the task.

Daily reminders will be sent out to everyone showing a list of all the tasks and issues in progress and any new tasks and issues that have been added throughout the day.

Tools and Technology

An appropriate computer is needed so that the software can be created. The computer needs to be able to handle light to moderate processing. Software editors such as Visual Studio, which is free, or Eclipse, which is open source, are needed to program the software. Any of these editors are suitable for creating this software solution. Once the software has been created, the program needs to be stored on computers so that it can be used.

Skills Required

The skills needed for this project include the ability to program/write code using a programming language. No other software or special hardware is needed for this project.

Finding the skills, software and hardware required is very feasible. A programming editor is needed for the project which can be obtained for free and in some instances open source. Finding the appropriate tools needed to program the entire software package can be searched online through websites, forums or through a programming tutorial.

Outcome

If the project becomes a successful software solution individuals, businesses and teams can use this software to become more organised in the way they work.

The original problems lies within an individual or group who struggle to keep track of or forget about tasks that need to be completed by a certain date. This software is a solution that may not necessarily eradicate the lack of awareness or lack or memory but it will definitely improve the organisation and awareness of what needs to be done and what issues have risen.

This development will improve the efficiency at which people work and it may have the likelihood of increasing profitability or decreasing the loss in profit that businesses will appreciate in the future.
